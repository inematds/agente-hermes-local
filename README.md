# 🔒 Agente Hermes + Ollama — Seu SO de IA 100% Privado

Curso prático do [INEMA.CLUB](https://inema.club) sobre rodar um **agente de IA completo e privado** na sua própria máquina, com **Ollama** — sem mensalidade, sem rate limit, funciona offline e nenhum dado sai de casa.

**Acesse:** https://inematds.github.io/agente-hermes-local/

## Trilhas

1. **Fundamentos** — por que IA local é o futuro, o vocabulário (LLM, agente, "SO de IA"), o que é o Ollama, janela de contexto/64k, o trade-off privacidade × performance × preço e os três modos (Vault / Connected / Cloud).
2. **Mão na massa** — instalar o Ollama, escolher o modelo certo pro seu hardware, baixar e conversar, o Qwen 3 Coder 64k, conectar o modelo local ao Agente Hermes e os caminhos app desktop × terminal × Telegram.
3. **Projetos** — 7 projetos passo a passo: chat privado local, agente Hermes em Vault, memória/personas/skills, GitHub e documentos, background agents 24/7 a $0, alternar entre os modos e Hermes no celular.

## Como abrir

Site estático, self-contained (HTML + Tailwind CDN + JS inline). Abra `index.html` no navegador ou sirva a pasta. Funciona offline.

## Recursos de aprendizagem

Marcar-lido por seção, progresso por módulo/trilha/curso, anotação e highlight no próprio texto, "minha jornada" (com export/import `.json`), temas (escuro/claro/sépia/foco/alto-contraste) e modo de leitura.
